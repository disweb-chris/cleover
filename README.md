# Cleover - Gestión de Proyectos para Clientes 🚀

**Cleover** es una aplicación web diseñada para ayudar a las agencias de desarrollo web a gestionar proyectos y tareas para sus clientes de manera eficiente. La aplicación permite realizar un seguimiento del progreso de los proyectos en tiempo real, ofreciendo un tablero intuitivo para los usuarios.

---

## 🛠️ **Tecnologías utilizadas**

- **Next.js** (Framework de React)
- **Firebase** (Autenticación, Firestore y Hosting)
- **TypeScript** (Tipado estático)
- **Vercel** (Despliegue en producción)

---

## 📋 **Funcionalidades principales**

- ✅ Pantalla de Login y Registro
- ✅ Protección de rutas privadas
- ✅ CRUD de Clientes
- ✅ CRUD de Proyectos
- ✅ Logout
- ✅ Despliegue en producción
